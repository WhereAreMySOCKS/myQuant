## 1. K 线是什么？

K 线（Candlestick，也叫蜡烛图）是技术分析中最基础的数据形式，每一根 K 线代表一段时间内价格变动的**摘要信息**。

### K 线的五要素（OHLCV）

| 字母 | 含义 | 例子 |
|------|------|------|
| **O** (Open) | 开盘价 | 2024-01-15 早上 9:30 的第一笔成交价 |
| **H** (High) | 最高价 | 当天价格的最高点 |
| **L** (Low) | 最低价 | 当天价格的最低点 |
| **C** (Close) | 收盘价 | 当天 15:00 的最后成交价 |
| **V** (Volume) | 成交量 | 当天总成交股数（或成交额） |

### 日线 K 线图形示意

```
价格轴
  │
1700┤    ──── 最高价 (H)
  │    │
1680┤    │  ← 上影线（最高价到实体上边的细线）
  │   ███  ← 阳线实体（收盘 > 开盘时为红色/阳线）
1660┤   ███  上边 = 收盘价 (C = 1670)
  │   ███  下边 = 开盘价 (O = 1660)
1650┤    │  ← 下影线（实体下边到最低价的细线）
  │    ──── 最低价 (L)
1640┤
  │
  └────────────────── 时间轴
      2024-01-15
```

> 收盘价 > 开盘价 → **阳线**（红色）：当天上涨
> 收盘价 < 开盘价 → **阴线**（绿色）：当天下跌

### 为什么我们主要用收盘价？

计算均线时，通常使用**收盘价**，原因：
1. 收盘价是当天最终确定的价格，反映了所有交易者的共识
2. 机构大资金通常在收盘时完成操作，收盘价更具代表性
3. 国际上的技术分析惯例也以收盘价为主

在 myQuant 的代码中，`df['收盘']` 列就是每日收盘价。

## 2. 移动平均线（MA）

### 2.1 什么是移动平均线？

移动平均线（Moving Average，简称 MA）就是**过去 N 天收盘价的算术平均值**，随时间"滑动"计算。

**计算公式：**

$$\text{MA}(N)_t = \frac{C_t + C_{t-1} + C_{t-2} + \cdots + C_{t-N+1}}{N}$$

其中：
- $C_t$ = 第 $t$ 天的收盘价
- $N$ = 均线周期（天数）

### 2.2 具体计算例子

假设贵州茅台最近 5 天的收盘价为：
| 日期 | 收盘价 |
|------|-------|
| 第1天 | 1600 |
| 第2天 | 1620 |
| 第3天 | 1580 |
| 第4天 | 1640 |
| 第5天 | 1610 |

$$\text{MA5} = \frac{1600 + 1620 + 1580 + 1640 + 1610}{5} = \frac{8050}{5} = 1610$$

第5天的 MA5 = **1610 元**。

到了第6天（假设收盘价 1650），MA5 变为：

$$\text{MA5} = \frac{1620 + 1580 + 1640 + 1610 + 1650}{5} = \frac{8100}{5} = 1620$$

这就是"移动"的含义——每新增一天数据，就删去最旧的一天，窗口向前移动。

### 2.3 用 pandas 计算均线

```python
import pandas as pd

close = pd.Series([1600, 1620, 1580, 1640, 1610, 1650])

ma5 = close.rolling(5).mean()
# 结果：前4个值为 NaN（数据不足），第5个值开始才有效

print(ma5.iloc[-1])  # 1620.0
```

这正是 `analyzer.py` 中的实现方式：

```python
# 来自 app/services/analyzer.py
ma5  = close.rolling(5).mean().iloc[-1]
ma20 = close.rolling(20).mean().iloc[-1]
ma120 = close.rolling(120).mean().iloc[-1]
ma250 = close.rolling(250).mean().iloc[-1]
```

## 3. 各周期均线的含义

不同周期的均线代表不同的"视角"：

| 均线 | 周期 | 俗称 | 代表的时间维度 | 主要用途 |
|------|------|------|-------------|---------|
| **MA5** | 5 天 | 周线 | 约 1 周 | 超短线方向，参考价值有限 |
| **MA10** | 10 天 | 半月线 | 约 2 周 | 短期趋势 |
| **MA20** | 20 天 | 月线 | 约 1 个月 | 短期趋势，技术派常用 |
| **MA60** | 60 天 | 季线 | 约 3 个月 | 中期趋势 |
| **MA120** | 120 天 | 半年线 | 约 6 个月 | 中长期趋势 |
| **MA250** | 250 天 | **年线** | 约 1 年 | 长期价值中枢，**myQuant 核心** |

> A 股每年大约有 250 个交易日（全年 365 天 - 周末 - 节假日 ≈ 250 天）。

### 短期 vs 长期均线的直觉理解

- **短期均线（MA5/10）**：对价格变化反应快，波动大，噪音多
- **长期均线（MA250）**：反应慢，但更平滑稳定，代表市场对该股票的"长期共识价值"

想象一条河流：
- 短期均线 = 水面的波纹，随风波动
- 长期均线 = 河床走向，缓慢稳定

## 4. 乖离率（BIAS）

### 4.1 什么是乖离率？

乖离率衡量的是**当前价格偏离均线的程度**。

**计算公式：**

$$\text{BIAS}(N) = \frac{\text{当前价} - \text{MA}(N)}{\text{MA}(N)}$$

或者用百分比表示：

$$\text{BIAS}(N)\% = \frac{\text{当前价} - \text{MA}(N)}{\text{MA}(N)} \times 100\%$$

### 4.2 具体计算例子

**例 1：超跌状态（买入信号区域）**

假设贵州茅台当前股价 **1600 元**，MA250 = **1750 元**：

$$\text{BIAS} = \frac{1600 - 1750}{1750} = \frac{-150}{1750} \approx -0.0857 = \mathbf{-8.57\%}$$

解读：当前价格比年线低了约 8.57%，处于**超跌**状态。

**例 2：超涨状态（卖出信号区域）**

假设贵州茅台当前股价 **2100 元**，MA250 = **1750 元**：

$$\text{BIAS} = \frac{2100 - 1750}{1750} = \frac{350}{1750} = 0.2 = \mathbf{+20\%}$$

解读：当前价格比年线高了 20%，处于**超涨**状态。

**例 3：贴近均线（中性区域）**

假设股价 **1770 元**，MA250 = **1750 元**：

$$\text{BIAS} = \frac{1770 - 1750}{1750} \approx 0.0114 = \mathbf{+1.14\%}$$

解读：价格略高于年线，基本正常，无明显信号。

### 4.3 乖离率在 analyzer.py 中的实现

```python
# 来自 app/services/analyzer.py
bias_rate = (current_price - ma250) / ma250

result = {
    "price": round(current_price, 3),
    "ma250": round(ma250, 3),
    "bias_rate": round(bias_rate, 4),        # 小数形式，如 -0.0857
    "bias_percent": f"{bias_rate:.2%}",      # 百分比字符串，如 "-8.57%"
}
```

## 5. 为什么选 MA250？

为什么 myQuant 选择 MA250 而不是其他均线？

### 5.1 MA250 ≈ 年线，意义特殊

A 股每年约 250 个交易日，MA250 反映的是**过去一整年的平均价格**。这意味着：
- 它包含了一个完整年度的市场行为
- 受季节性因素、年报发布、分红等长周期事件的综合影响
- 被大量机构投资者视为重要的支撑/阻力位

### 5.2 长均线比短均线稳定

短期均线（MA5）可能因为一天的大幅波动就产生信号，误报率高。MA250 变化缓慢，只有趋势性变化才会触发阈值，减少了"假信号"。

### 5.3 参数适合回测验证

用更短的均线（如 MA20）会产生非常多的交易信号（频繁买卖），手续费影响显著，且更依赖参数的精确调优（过拟合风险高）。MA250 的信号频率适中，回测结果更具参考价值。

## 6. 均线的经典用法

除了乖离率，均线还有一些经典的使用方式：

### 6.1 金叉与死叉

当两条不同周期的均线交叉时，被认为是趋势转换的信号：

- **金叉**（黄金交叉）：短期均线从下方向上穿越长期均线 → **看涨信号**
  - 例：MA5 上穿 MA20
- **死叉**（死亡交叉）：短期均线从上方向下穿越长期均线 → **看跌信号**
  - 例：MA5 下穿 MA20

```
价格
│      /\
│     /  \   ← MA5（短均线）
│    /    \__/
│   /  金叉  死叉
│  /─────────────── MA20（长均线）
└────────────────── 时间
```

### 6.2 多头排列与空头排列

当多条均线从上到下按短期到长期排列时：

- **多头排列**：MA5 > MA10 > MA20 > MA60 > MA120 > MA250 → 强势上涨格局
- **空头排列**：MA5 < MA10 < MA20 < MA60 < MA120 < MA250 → 持续下跌格局

### 6.3 均线作为支撑/阻力位

- 股价在均线附近往往会出现**支撑**（跌到均线附近止跌反弹）
- 股价从下方接近均线时，均线可能成为**阻力**（涨到均线附近受阻回落）

MA250 因为代表大量投资者的平均持仓成本，对这个效应尤为明显。

## 7. 乖离率的经典用法

乖离率除了触发买卖信号，还有其他用途：

### 7.1 超买/超卖判断

| 乖离率值 | 市场状态 | 操作含义 |
|---------|---------|---------|
| > +20% | 严重超涨 | 注意风险，考虑减仓 |
| +10% ~ +20% | 超涨 | 谨慎追高 |
| -5% ~ +10% | 正常区间 | 无明显信号 |
| -10% ~ -5% | 超跌 | 开始关注 |
| < -10% | 严重超跌 | 分批建仓区间 |

### 7.2 注意：不同股票的正常乖离率不同

高波动股票（如科技股）的正常乖离率范围比低波动股票（如银行股）要大得多。因此，-8% 的阈值对某些股票可能合理，对另一些股票可能太松或太紧。

**建议**：通过回测找到每只股票的最优参数（详见[第四章：回测入门与实践](04-backtesting-guide.md)）。

## 8. analyzer.py 完整代码解析

现在我们来完整看一遍 `app/services/analyzer.py`：

```python
def compute_indicators(df: pd.DataFrame, current_price: float, code: str = ""):
    """
    计算技术指标: MA5, MA20, MA120, MA250, 乖离率
    用实时价覆盖最后一行，盘中更准确。
    """
    # 1. 检查数据是否足够（至少 250 行）
    if df is None or len(df) < MIN_HISTORY_ROWS:  # MIN_HISTORY_ROWS = 250
        return None

    # 2. 取收盘价列，用实时价覆盖最后一行
    #    这样盘中查询时，均线计算会包含当前最新价格
    close = df['收盘'].copy()
    close.iloc[-1] = current_price

    # 3. 计算各均线（pandas rolling mean）
    ma5  = close.rolling(5).mean().iloc[-1]
    ma20 = close.rolling(20).mean().iloc[-1]
    ma120 = close.rolling(120).mean().iloc[-1]
    ma250 = close.rolling(250).mean().iloc[-1]

    # 4. 防御性检查：任意均线为 NaN 则放弃（数据有缺口）
    if any(math.isnan(v) for v in (ma5, ma20, ma120, ma250)):
        return None

    # 5. 计算 MA250 乖离率
    bias_rate = (current_price - ma250) / ma250

    return {
        "price": round(current_price, 3),
        "ma5": round(ma5, 3),
        "ma20": round(ma20, 3),
        "ma120": round(ma120, 3),
        "ma250": round(ma250, 3),
        "bias_rate": round(bias_rate, 4),
        "bias_percent": f"{bias_rate:.2%}",
    }
```

**关键设计点：**
- `close.iloc[-1] = current_price`：用实时价替换最后一行收盘价，使得盘中计算的均线也包含最新价格，更准确地反映当前状态
- `MIN_HISTORY_ROWS = 250`：严格要求至少 250 行数据，确保 MA250 有效
- 乖离率以小数（-0.0857）和百分比字符串（"-8.57%"）两种形式返回，方便不同场景使用
