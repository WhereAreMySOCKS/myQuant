## 1. 为什么风险管理比策略更重要？

很多初学者认为：找到一个好策略 = 成功。

但实际上，**风险管理是量化交易中最重要的环节**，没有之一。

### 1.1 一个破产的例子

假设你有 100,000 元，使用一个胜率 60% 的策略：
- 每次赢：+10%（盈利 10,000 元）
- 每次输：-20%（亏损 20,000 元）

即使胜率 60%，期望收益是：

$$0.6 \times 10\% + 0.4 \times (-20\%) = 6\% - 8\% = -2\%$$

每次下注的**期望是亏损**，再高的胜率也救不了这个策略。

更可怕的是，如果你全仓操作，连续亏损 4 次：

$$100,000 \times (1 - 20\%)^4 = 100,000 \times 0.4096 = 40,960 \text{ 元}$$

亏损近 60%，心态可能早就崩溃了。

### 1.2 "活下去"才是第一原则

沃伦·巴菲特的两条投资规则：
1. 永远不要亏损
2. 永远不要忘记第一条

不亏损不是真正做得到的，但这说明了**保住本金**的重要性：
- 亏损 50% 需要盈利 **100%** 才能回本
- 亏损 80% 需要盈利 **400%** 才能回本
- 一次大亏损，可能需要很多年才能恢复

```
亏损幅度 → 需要的回本收益率
-10%    → +11.1%
-20%    → +25%
-30%    → +42.9%
-50%    → +100%
-80%    → +400%
```

## 2. 仓位管理基础

### 2.1 什么是仓位管理？

仓位管理（Position Sizing）决定的是：**每次买入应该用多少钱？**

即使策略信号触发，也不意味着要把所有钱都投进去。

### 2.2 三种常见的仓位策略

**① 固定仓位法**（最简单）

每次只买总资金的固定比例，如 30%：
- 总资金 10 万，每次只买 3 万
- 优点：简单可控
- 缺点：没有充分利用资金

**② 全仓法**（myQuant 当前使用）

每次买入时用全部可用资金：
- 优点：信号触发时利润最大化
- 缺点：单次错误损失大，无法摊平成本

**③ 金字塔加仓法**（进阶）

第一次买入 1/3 仓，如果继续下跌再加 1/3，再跌再加 1/3：

```
价格
│
100┤
   │                ●  第一批买入（1/3仓，乖离率 -5%）
 95┤              ╱
   │            ╱
 90┤          ●  第二批买入（再加 1/3仓，乖离率 -10%）
   │        ╱
 85┤      ●  第三批买入（再加 1/3仓，乖离率 -15%）
   │    ╱
 80┤  ╱
   └───────────────────── 时间
```

优点：降低了平均成本，更稳健
缺点：如果价格持续下跌，三批全满仓后仍然亏损

### 2.3 凯利公式（数学最优仓位）

凯利公式给出了在已知胜率和赔率下的最优仓位比例：

$$f^* = \frac{p \cdot b - q}{b}$$

其中：
- $f^*$ = 最优仓位比例
- $p$ = 胜率
- $q = 1 - p$ = 败率
- $b$ = 盈亏比（平均盈利 / 平均亏损）

**例子**：胜率 60%，盈亏比 2：1（赢 +20%，输 -10%）

$$f^* = \frac{0.6 \times 2 - 0.4}{2} = \frac{1.2 - 0.4}{2} = \frac{0.8}{2} = 40\%$$

最优仓位是 40%（不是全仓！）。

> 💡 在实际操作中，建议使用"半凯利"（20%），因为凯利公式的假设在现实中难以完全满足，过于激进会导致大幅回撤。

## 3. 止损和止盈

### 3.1 止损（Stop Loss）

止损是预先设定一个**亏损上限**，亏到这个价格就无条件卖出。

**例子**：买入价 1600 元，设 -10% 止损，即跌到 1440 元就卖出，不管后续如何。

**为什么需要止损？**

1. **控制单笔最大损失**：防止一次错误买入毁掉整体策略
2. **防止"鸵鸟心态"**：很多人亏损后不愿割肉，越套越深

**止损 vs 均值回归策略的矛盾**

这里有一个值得思考的矛盾：
- 均值回归策略认为"跌多了会涨回来"
- 止损策略认为"亏损太多就割肉"

这两个逻辑在某种程度上是相反的。

**解决方案**：设置基于**时间**的止损（比如持有超过 3 个月没有回归，割肉离场），而不是纯粹基于价格的止损。或者对**单只股票**设置价格止损，对**整体组合**设置最大回撤止损。

### 3.2 止盈（Take Profit）

止盈是设定一个**盈利目标**，涨到这个价格就卖出。

myQuant 的卖出逻辑（乖离率 ≥ +15% 时卖出）本质上就是一种**动态止盈**：
- 随着 MA250 的移动，止盈价格也在变化
- 不是固定的"涨了 X% 就卖"，而是"相对年线贵了 X% 才卖"

## 4. 最大回撤控制

### 4.1 回撤与人性

从心理学角度，投资者对**已经盈利后再亏损**的痛苦，远大于相同金额的绝对亏损。

- 账户从 10 万涨到 15 万，再跌回 12 万 → 明明还是赚的，但感觉亏了 3 万
- 这种回撤会让人产生"我本可以更多"的懊悔，导致不理性决策

### 4.2 不同最大回撤的心理影响

| 最大回撤 | 心理影响 | 大多数人的反应 |
|---------|---------|-------------|
| -5% | 几乎没感觉 | 轻松持有 |
| -10% | 轻微担忧 | 偶尔查看 |
| -15% | 开始焦虑 | 频繁查看，考虑离场 |
| -20% | 明显恐慌 | 很多人选择离场 |
| -30% | 极度恐慌 | 大多数人割肉离场 |
| -50% | 绝望 | 几乎所有人已经离场 |

> 如果你的策略历史最大回撤是 -30%，你要问自己：**当账户从 10 万跌到 7 万时，我能坚持持有吗？**

### 4.3 最大回撤上限的设定建议

在开始实盘前，先问自己：**我能承受的最大亏损是多少？**

- 保守型：最大回撤不超过 -10%
- 稳健型：最大回撤不超过 -20%  
- 激进型：最大回撤不超过 -30%

如果回测显示策略的最大回撤超过了你的承受上限，就需要调整策略（比如加入止损或仓位控制），而不是在实盘中强迫自己忍受。

## 5. myQuant 中的风险管理实现

### 5.1 阈值报警作为风险防线

myQuant 的报警系统是第一道防线：当价格触发阈值时，不是自动交易，而是**发邮件通知人工判断**。

这样保留了人工干预的空间：
- 如果市场整体在暴跌（熊市），可以选择不执行买入信号
- 如果某公司突发利空（重大负面事件），可以选择不买入

### 5.2 报警去重

`AlertStateManager` 确保同一标的同方向一天只报警一次，避免在价格围绕阈值反复震荡时被邮件轰炸。

```python
# 来自 cache.py（AlertStateManager 设计思路）
# 报警记录结构：{code: {direction: date}}
# 每天首次触发才发邮件，当天后续触发忽略
```

### 5.3 监控并发控制

`MONITOR_CONCURRENCY` 配置项控制同时处理的标的数量，防止短时间内对数据源发起太多请求（被封 IP）：

```python
# core/config.py
MONITOR_CONCURRENCY: int = 5  # 每批最多并发处理 5 个标的
```

### 5.4 回测中的风险指标

回测结果中包含的 `max_drawdown` 字段是评估策略风险的重要依据：

```json
"summary": {
  "max_drawdown": -0.1821,           // -18.21%
  "max_drawdown_pct": "-18.21%",
  ...
}
```

在选择策略参数时，应该将最大回撤纳入重要的评估标准，而不仅仅追求最高收益率。

## 6. 情绪管理和纪律的重要性

### 6.1 量化交易 vs 手动交易的核心优势

| 方面 | 主观交易 | 量化交易 |
|------|---------|---------|
| 执行纪律 | 受情绪影响，容易破坏规则 | 规则固定，自动执行 |
| 决策速度 | 反应较慢，可能错过信号 | 实时监控，毫秒响应 |
| 同时处理标的数 | 有限（人脑处理能力） | 可以同时监控几十上百个 |
| 过去表现验证 | 难以客观评估 | 可以精确回测 |
| 情绪影响 | 恐慌时追跌杀、贪婪时追高 | 无情绪，严格按规则执行 |

**量化交易最大的价值，不是找到"神奇公式"，而是通过规则化的执行，消除人性弱点的干扰。**

### 6.2 常见的量化交易情绪陷阱

即使是量化策略，人也会在以下场景失去纪律：

1. **策略暂时亏损时**：忍不住关闭策略，然后策略就开始盈利了
2. **策略暂时大赚时**：忍不住加大仓位，恰好遇到大回撤
3. **频繁优化参数**：每次看到回撤就修改参数，导致过拟合
4. **对别人的策略眼红**：频繁切换策略，却总是在高位入场

### 6.3 建立纪律的方法

1. **书面化策略规则**：把策略的所有规则写下来，包括什么情况下允许人工干预
2. **定期复盘（不是随时复盘）**：每周或每月看一次结果，避免被日常波动影响情绪
3. **合理的仓位**：只用你心态上能承受全部损失的钱来做量化
4. **足够长的验证期**：给策略至少 1 年的实盘时间，不要因为 1 个月的亏损就放弃

## 7. 风险管理清单

在开始实盘之前，对照以下检查清单：

- [ ] 是否通过回测验证了策略在不同市场环境下的表现？
- [ ] 策略的历史最大回撤是否在你的心理承受范围内？
- [ ] 是否只用你能承受全部损失的资金做量化？
- [ ] 是否设置了止损规则（或明确决定不设置及其理由）？
- [ ] 是否理解了策略的适用场景和不适用场景？
- [ ] 是否有书面的策略执行规则，避免临时改变主意？
- [ ] 是否理解了回测可能过于乐观（忽略手续费、滑点等）？
